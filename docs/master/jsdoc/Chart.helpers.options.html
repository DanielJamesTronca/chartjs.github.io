Methods

<static> resolve(inputs [, context] [, index] [, info])

Evaluates the given `inputs` sequentially and returns the first defined value.
Parameters:
Name Type Argument Description
inputs Array An array of values, falling back to the last value.
context object <optional>
 If defined and the current value is a function, the value is called with `context` as first argument and the result becomes the new input.
index number <optional>
 If defined and the current value is an array, the value at `index` become the new input.
info object <optional>
 object to return information about resolution in
Properties
Name Type Argument Description
cacheable boolean <optional>
 Will be set to `false` if option is not cacheable.
Since:
  • 2.7.0

<static> toLineHeight(value, size)

Converts the given line height `value` in pixels for a specific font `size`.
Parameters:
Name Type Description
value number | string The lineHeight to parse (eg. 1.6, '14px', '75%', '1.6em').
size number The font size (in pixels) used to resolve relative `value`.
Since:
  • 2.7.0
See:
  • https://developer.mozilla.org/en-US/docs/Web/CSS/line-height
Returns:
The effective line height in pixels (size * 1.2 if value is invalid).
Type
number

<static> toPadding(value)

Converts the given value into a padding object with pre-computed width/height.
Parameters:
Name Type Description
value number | object If a number, set the value to all TRBL component, else, if and object, use defined properties and sets undefined ones to 0.
Since:

Returns:
The padding values (top, right, bottom, left, width, height)
