Class: exports

exports

Abstract class that allows abstracting platform dependencies away from the chart.

new exports()

Members

disableCSSInjection

When `true`, prevents the automatic injection of the stylesheet required to correctly detect when the chart is added to the DOM and then resized. This switch has been added to allow external stylesheet (`dist/Chart(.min)?.js`) to be manually imported to make this library compatible with any CSP. See https://github.com/chartjs/Chart.js/issues/5208

<static> notify(chart, hook [, args])

Calls enabled plugins for `chart` on the specified hook and with the given args. This method immediately returns as soon as a plugin explicitly returns false. The returned value can be used, for instance, to interrupt the current action.
Parameters:
Name Type Argument Description
chart Chart The chart instance for which plugins should be called.
hook string The name of the plugin method to call (e.g. 'beforeUpdate').
args Array <optional>
 Extra arguments to apply to the hook call.
Returns:
false if any of the plugins return false, else returns true.

boolean
